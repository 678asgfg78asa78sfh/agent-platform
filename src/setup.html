⚡ Agent Platform — Setup

Willkommen! Um deinen ersten Agent zu bauen brauchst du ein LLM-Backend. Wähl eine der Optionen unten, test, speicher — dauert ~30 Sekunden. Danach steht der KI-Wizard bereit und du erstellst Agenten indem du mit ihm redest.

🌐 OpenRouter

FREI · viele Modelle
OpenRouter ist ein LLM-Aggregator mit dutzenden Modellen (GPT, Claude, Gemini, DeepSeek, Qwen, Grok, Kimi) hinter einer API. Freie Modelle verfügbar (z.B. deepseek/deepseek-chat:free, qwen/qwen3-coder:free), kostenpflichtige Top-Modelle on-demand. API-Key holen auf openrouter.ai/keys (braucht nur Email).

🦙 Ollama (lokal)

KOSTENLOS · läuft lokal
Falls ollama serve auf diesem Rechner läuft: null Setup nötig, voll privat, keine Cloud. Modell vorab ziehen: ollama pull llama3.2. Siehe ollama.com.

🤖 OpenAI

BEZAHLT · GPT-5.x
Standard OpenAI API. Key auf platform.openai.com/api-keys.

🧠 Anthropic (Claude)

BEZAHLT · Claude 4.x
Direkt bei Anthropic. Key auf console.anthropic.com.

🐫 llama.cpp server

LOKAL · OpenAI-kompatibel
Falls ./llama-server läuft (OpenAI-kompatibel auf :8080). Passt für Power-User mit eigenen GGUF-Modellen.
Fortgeschrittene können auch direkt agent-data/config.json editieren — siehe docs/templates/. Nach dem Setup bist du auf / (Admin-UI) und kannst den KI-Wizard starten.
