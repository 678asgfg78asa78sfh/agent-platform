Welcome! To build your first agent you need an LLM backend. Pick an option below, test it, save — takes about 30 seconds. After that the AI Wizard is ready and you create agents by talking to it.

🌐 OpenRouter

FREE · many models
OpenRouter is an LLM aggregator with dozens of models (GPT, Claude, Gemini, DeepSeek, Qwen, Grok, Kimi) behind one API. Free models available (e.g. deepseek/deepseek-chat:free, qwen/qwen3-coder:free), paid flagship models on demand. Get an API key at openrouter.ai/keys (email only, no card required for free tier).

🦙 Ollama (local)

FREE · runs locally
If ollama serve runs on this machine: zero setup, fully private, no cloud. Pull a model first: ollama pull llama3.2. See ollama.com.

🤖 OpenAI

PAID · GPT-5.x
Standard OpenAI API. Get a key at platform.openai.com/api-keys.

🧠 Anthropic (Claude)

PAID · Claude 4.x
Direct Anthropic API. Get a key at console.anthropic.com.

🐫 llama.cpp server

LOCAL · OpenAI-compatible
If ./llama-server runs (OpenAI-compatible on :8080). For power users running their own GGUF models.
Advanced users can edit agent-data/config.json directly — see docs/templates/. After setup you land on / (admin UI) and can start the AI Wizard.
